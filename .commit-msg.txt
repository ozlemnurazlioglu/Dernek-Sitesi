feat(home): SMS aboneligi formu, admin liste + CSV (Excel) indir

Ana sayfaya KVKK onayli, uyelik gerektirmeyen SMS abone formu eklendi;
toplanan numaralar admin panelinden yonetilebilir.

- DB: yeni `sms_subscribers` tablosu (UNIQUE phone, KVKK onay zamani,
  IP/UA). Migration: scripts/add-sms-subscribers.ts
- Public API: POST /api/sms-subscribers (auth gerektirmez); TR cep
  numarasi normalize, KVKK onayi zorunlu, ayni numara icin 409
  ALREADY_SUBSCRIBED.
- Public UI: ana sayfaya yeni `sms_subscribe` blok (HomeLayout
  sistemi ile entegre, admin layout'tan tasinabilir / gizlenebilir).
  Form: numara input + KVKK checkbox + Abone Ol; KVKK metni
  /kvkk sayfasina link; sonuc inline banner olarak gosterilir.
- Admin /admin/sms-aboneleri: liste, tek tek silme, "Excel olarak
  indir" butonu (UTF-8 BOM CSV, ; ayrac, telefon hucresi metin
  olarak). Sidebar'a "SMS Aboneleri" linki eklendi.
- Admin Sayfa Icerikleri > Ana Sayfa: yeni "SMS Aboneligi" editor
  ile basliklar/KVKK metni/durum mesajlari ozellestirilebilir.
- Type/seed: HomeBlockId'ye `sms_subscribe`, HomeSmsSubscribeBlock
  ve SmsSubscriber tipleri, default seed ve default home layout
  guncellendi.
